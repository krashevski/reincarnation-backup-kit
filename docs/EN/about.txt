Reincarnation Backup Kit (REBK)

Reincarnation Backup Kit is a set of Bash scripts for backup and restore when reinstalling Ubuntu on an SSD, as well as for creating a multimedia environment (Shotcut, GIMP+G'MIC, Krita, Audacity).

Modular shell-based backup and restore toolkit designed for GNU/Linux systems. Primarily tested on systemd-based distributions.

The project focuses on:
  • transparency
  • user control
  • system recoverability
  • minimal dependencies

REBK operates strictly on user-owned data and system
configuration files.

REBK does not include or distribute proprietary software.
It operates exclusively on user-owned data.
